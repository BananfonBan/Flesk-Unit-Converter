This is a simple unit converter written like Flask-application.

The application is made as an example project from [roadmap.sh](https://roadmap.sh/projects/unit-converter)

### Requirements
- python
- poetry
- flask
- jinja2
- gunicorn

### Setup
```bash
make setup
```

### Run server
```bash
make start
```
